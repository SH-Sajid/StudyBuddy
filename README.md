# 🤖 StudyBuddy

**Your AI-Powered Document Q&A Assistant**

Ask **ANY question** about your documents and get instant, sourced answers!

## 🚀 Features

- **📚 Document Processing**: Upload PDF/TXT files or paste text directly
- **❓ Any Question Support**: Ask factual, explanatory, analytical, or comparative questions
- **🎯 Accurate Answers**: AI-powered responses with confidence scores
- **🔍 Source Evidence**: See exactly where answers come from in your document
- **📄 Smart Summarization**: Generate concise document summaries
- **💡 Question Suggestions**: Get help with alternative question formats

## 📖 How to Use

1. **Add Content**: Upload a PDF/TXT file OR paste text directly
2. **Process Document**: Click "Analyze Document" to prepare for questions
3. **Ask Anything**: Type your question about the document content
4. **Get Answers**: Receive AI-powered responses with source evidence

## 💡 Example Questions

- "What is the main topic of this document?"
- "Explain the key concepts mentioned"
- "What methods were used in the research?"
- "List the important findings"
- "Compare the different approaches discussed"

## 🛠️ Technology

- **Gradio**: Beautiful web interface
- **Transformers**: Hugging Face AI models for Q&A and summarization
- **scikit-learn**: TF-IDF for document retrieval
- **PyPDF2/pdfplumber**: PDF text extraction
